# Profile Photo

Please add Kyle Montague's profile photo as `profile.jpg` in this directory.

Recommended specifications:
- Format: JPG or PNG
- Size: At least 400x400 pixels (square)
- File size: Under 500KB for fast loading

If you want to use a different filename or format:
1. Name your file accordingly (e.g., `profile.png`)
2. Update the image source in `index.html`:
   - Find: `<img src="profile.jpg" alt="Kyle Montague" ...>`
   - Change to: `<img src="profile.png" alt="Kyle Montague" ...>`

The CSS will automatically resize and crop the image into a circular shape.
